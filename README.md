# Morphkit

Semantic AI agent that converts TypeScript/React web apps to native SwiftUI iOS apps.

## Quick Start

```bash
bun install
bun run src/index.ts analyze ./path-to-nextjs-app
bun run src/index.ts generate ./path-to-nextjs-app --output ./ios-app
```

## Requirements

- Bun 1.0+
- Node.js 20+ (alternative)
- XAI_API_KEY (optional, for AI-enhanced analysis)
